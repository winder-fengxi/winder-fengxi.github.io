# winder-fengxi.github.io

这是一个静态文件存储仓库，用于通过 GitHub Pages 分发静态资源。

## 目录结构

- **[images/](images/)**: 存放图片文件 (jpg, png, gif, svg 等)
- **[css/](css/)**: 存放样式表文件 (css, scss 等)
- **[js/](js/)**: 存放脚本文件 (javascript)
- **[files/](files/)**: 存放其他类型文件 (pdf, doc, zip 等)

## 使用方法

1. 将文件上传到对应目录。
2. 通过 GitHub Pages URL 访问资源，例如：
   `https://winder-fengxi.github.io/images/your-image.jpg`
